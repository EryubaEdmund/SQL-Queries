# SQL Practice – Intermediate Queries and Data Analytics

## 📌 Project Overview

This repository contains a collection of PostgreSQL queries demonstrating advanced SQL concepts using sample database tables such as:
- customers
- products
- sales
- inventory

The queries focus on real-world analytical scenarios including aggregation, joins, filtering, grouping, self-joins, and date-based analysis.

## 🎯 Objectives

The exercises demonstrate how to:

1. Perform table joins (INNER JOIN, self-join)
2. Use aggregate functions (SUM, AVG, COUNT, MAX, MIN)
3. Apply date filtering and time-based grouping
4. Identify top-performing customers and products
5. Analyze sales trends by month and year
6. Work with stock and inventory data
7. Apply filtering with conditions such as price ranges and string matching

## 🛠 Technologies Used

- PostgreSQL
- SQL (Structured Query Language)
- pgAdmin / psql
- Relational Database Concepts

## 🚀 How to Use

1. Create the tables in PostgreSQL.
2. Insert sample data.
3. Execute the queries individually in psql, pgAdmin, or any SQL client.
4. Modify and experiment with additional analytical queries.
